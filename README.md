## instruction
- Download the [ZIP](https://github.com/bitwire-it/bitwarden-id-extractor/archive/refs/heads/main.zip) and extract the folder
- Open Google Chrome and go to chrome://extensions/.
- Enable Developer mode using the toggle switch in the top right corner.
- Click the Load unpacked button that appears.
- Select your bw-id-extractor folder.

## How to Use
- Open your Bitwarden Web Vault (vault.bitwarden.com).
- Click on an item to view/edit it. Ensure the URL looks something like ...itemId=xxxxx....
- Click the extension icon in your toolbar.
- The ID will be automatically copied to your clipboard!
